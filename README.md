# Patrick Estrella — Full-Stack Developer Portfolio

A modern, minimal streetwear-inspired portfolio showcasing my work as a 4th-year BSIT student and full-stack developer.

**Live at:** [http://localhost:3001](http://localhost:3001)

## 🎨 Design

Built with a **clean, bold aesthetic** inspired by streetwear culture:
- **Typography:** Anton (display) + Space Grotesk (body) + Caveat (accents)
- **Color Palette:** Warm neutrals with muted tan accents
- **Features:**
  - Fullscreen intro animation with smooth video transition
  - Minimal card-based project showcase
  - Timeline-based education & experience section
  - Responsive design powered by clamp() and CSS Grid

## 🛠 Tech Stack

- **Framework:** Next.js 16.2.2 (Turbopack)
- **Language:** TypeScript/TSX
- **Styling:** Tailwind CSS + inline styles
- **Fonts:** Google Fonts (Anton, Space Grotesk, Caveat)
- **Deployment Ready:** Next.js static export

## 📁 Project Structure

```
src/app/
├── page.tsx       → Main portfolio component with all sections
├── layout.tsx     → Root layout with font imports & metadata
└── globals.css    → Design tokens & animations
```

## ✨ Sections

1. **Hero** — Fullscreen intro with animated chibi character video
2. **Work** — 3 featured projects with hover effects
3. **About** — Bio, stats grid, and education timeline
4. **Contact** — CTA with email & social links
5. **Footer** — Minimal branding

## 🚀 Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view.

## 📦 Build for Production

```bash
npm run build
```
